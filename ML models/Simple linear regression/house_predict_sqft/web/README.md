# Simple Linear Regression Web App

A simple linear regression web application built with:

- 📦 **Backend**: Python Flask  
- 🎨 **Frontend**: Vite + React + Tailwind CSS  
- 📈 **Model**: One input → One output (Simple Linear Regression)
